Overview

A collection is a component used to display multiple records in an aggregate view. By default, collections offer a table view and a card view of your data. These views can be turned of or off based on the component's configuration.

Collections can be configured with static data or dynamically loaded from a remote datasource. If your remote datasource can deliver data in a paged result and the collection can load data iteratively with an infinite scroll behavior.

Example

{% highlight javascript %}
import { Collection } from 'reframe';

class Index extends React.Component {

  render() {
    return 
  }

  _getCollection() {
    return {
      id: 'people',
      columns: [
        { label: 'First Name', key: 'first_name', visible: true },
        { label: 'Last Name', key: 'last_name', visible: true }
      ],
      records: [
        { first_name: 'Greg', last_name: 'Kops' },
        { first_name: 'Megan', last_name: 'Pugh' },
        { first_name: 'Kath', last_name: 'Tibbetts' },
        { first_name: 'Armand', last_name: 'Zerilli' }
      ]
    }
  }

}

export default Index
{% endhighlight %}

Attributes

A collection can be configured with the following attributes:

{% highlight javascript %}
{
  id: "string",
  columns: "string/array",
  records: "string/array",
  filters: "string/array",
  sort: "object",
  card: "object",
  entity: "string",
  recordActions: "array",
  batchActions: "array"
}
{% endhighlight %}

id

This is a required field and must be unique on any given page. It is a best practice to make sure that this value is globally unique across your application.

{% highlight javascript %}
id: 'contacts-index',
{% endhighlight %}

columns

Columns can be loaded asynchronously from a external datasource:

{% highlight javascript %}
columns: '/admin/customers/columns'
{% endhighlight %}

Columns can be also be loaded synchronously from a static object:

{% highlight javascript %}
columns: [
  { label: 'First Name', key: 'first_name', primary: true, visible: true },
  { label: 'Last Name', key: 'last_name', primary: true, visible: true }
]
{% endhighlight %}

records

Records can be loaded asynchronously from a external datasource:

{% highlight javascript %}
records: '/admin/contacts',
{% endhighlight %}

Records can be also be loaded synchronously from a static object:

{% highlight javascript %}
records: [
  { first_name: 'Greg', last_name: 'Kops' },
  { first_name: 'Megan', last_name: 'Pugh' },
  { first_name: 'Kath', last_name: 'Tibbetts' },
  { first_name: 'Armand', last_name: 'Zerilli' }
]
{% endhighlight %}

filters

Filters can be loaded asynchronously from a external datasource:

{% highlight javascript %}
filters: '/admin/contacts/fields',
{% endhighlight %}

Filters can be also be loaded synchronously from a static object:

{% highlight javascript %}
filters: [
  {
    label: 'Basic Information',
    fields: [
      { code: 'first_name', label: 'First Name', type: 'textfield' },
      { code: 'last_name', label: 'Last Name', type: 'textfield' },
    ]
  }
],
{% endhighlight %}

sort

{% highlight javascript %}
sort: {
  key: 'created_at',
  order: 'desc'
}
{% endhighlight %}

card

Filters can be loaded asynchronously from a external datasource:

{% highlight javascript %}
card: {
  image: 'photo',
  url: '/admin/contacts/#{id}',
  content: ContentCard
}
{% endhighlight %}

entity

Filters can be loaded asynchronously from a external datasource:

{% highlight javascript %}
entity: 'contact'
{% endhighlight %}

empty

Filters can be loaded asynchronously from a external datasource:

{% highlight javascript %}
empty: 'There are no contacts'
{% endhighlight %}

recordActions

Filters can be loaded asynchronously from a external datasource:

{% highlight javascript %}
recordActions: [
  {
    label: 'edit',
    icon: 'edit',
    redirect: '/admin/contacts/#{id}/edit'
  }
]
{% endhighlight %}

batchActions

Filters can be loaded asynchronously from a external datasource:

{% highlight javascript %}
batchActions: [
  {
    label: 'delete all',
    component: AddToList
  }
]
{% endhighlight %}
